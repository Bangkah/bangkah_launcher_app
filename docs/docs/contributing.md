# Contributing

## Ways to Contribute
- Report bugs (issues)
- Suggest features
- Submit PRs
- Improve docs

## Workflow
```bash
git checkout -b feature/amazing-feature
# make changes, add tests, update docs
./vendor/bin/phpunit
git commit -m "Add amazing feature"
git push origin feature/amazing-feature
# open PR
```

## Code of Conduct
Respek, inklusif, bantu pendatang baru, feedback konstruktif.
