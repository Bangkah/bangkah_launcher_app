# Templates

## Web Template
Includes:
- `HomeController`
- `resources/views/home.blade.php`
- `routes/web.php`

Homepage: modern, responsive, Vite-ready.

Use cases: e-commerce, business sites, CMS, admin dashboards, marketing.

## API Template
Includes:
- `Api/HealthController`
- `routes/api.php`
- `bootstrap/app.php` (API routes enabled)

Health Endpoint:
`GET /api/health` returns status, app, env, versions, time.

Features: CORS, versioning ready, JSON responses, rate limiting, exception handling.
