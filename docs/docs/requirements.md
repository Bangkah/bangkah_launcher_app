# Requirements

| Requirement | Version | Notes |
| --- | --- | --- |
| PHP | 8.2+ | Laravel 12.x |
| Composer | 2.0+ | Dependency management |
| Node.js | 18+ / 20+ LTS | Frontend assets |
| NPM | 9+ | Package manager |
| Docker | 20+ | Optional |
| Docker Compose | 2.0+ | Optional |
| Git | 2.x+ | Recommended |

## System Requirements

- Minimum: 2 CPU, 2GB RAM, 5GB disk
- Recommended: 4+ CPU, 4GB+ RAM, 10GB+ disk (with Docker)
