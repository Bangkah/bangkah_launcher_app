# CLI

Referensi perintah Command Line Interface untuk Bangkah Launcher.

## Bantuan

```bash
bangkah-launcher --help
```

## Perintah Umum

- `init`: Inisialisasi proyek baru.
- `serve`: Menjalankan server pengembangan.
- `build`: Membangun artefak produksi.
- `config`: Menampilkan atau mengubah konfigurasi.

Contoh:

```bash
bangkah-launcher init my-app
bangkah-launcher serve --port 8000
bangkah-launcher build
bangkah-launcher config --set APP_DEBUG=0
```

Catatan: Sesuaikan daftar perintah dengan implementasi aktual di repositori.
