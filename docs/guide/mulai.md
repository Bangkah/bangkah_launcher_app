# Mulai

Dokumentasi ini membantu Anda menyiapkan dan menggunakan Bangkah Launcher.

## Prasyarat

- Node.js 18+ (untuk menjalankan dokumentasi, opsional)
- Git (untuk clone repositori)
- Sistem operasi Linux/macOS/Windows

## Clone Proyek

```bash
git clone https://github.com/Bangkah/bangkah-launcher.git
cd bangkah-launcher
```

## Instalasi

Instruksi instalasi spesifik Bangkah Launcher akan disesuaikan dengan repositori. Jika tersedia skrip `install` atau rilis binary, ikuti petunjuk di README proyek.

Contoh umum:

```bash
# Jika menggunakan npm untuk utilitas CLI
npm install -g bangkah-launcher

# Atau jalankan langsung dari source (sesuaikan)
./bin/bangkah-launcher --help
```

Lanjutkan ke [Instalasi](./instalasi) untuk detail tiap platform.
